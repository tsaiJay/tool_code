## multi_cam_recorder

### 功能說明
提供一部電腦上同步錄製多部webcam畫面的程式
```
$ pip install opencv
```
### 使用方法
- 更改儲存資料夾路徑
- 設定要錄製的webcam ID
```
    SAVE_DIR = './shoot_multi_vid/'
    CAM_ID = [0, 1]
```
